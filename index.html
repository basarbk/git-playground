Git Cheat Sheet

Config
git config - configure git system wide, global or local
Create
git init - create empty git repository in current directory
git clone - clone remote repository
Workflow
git add - add file to staging area
git commit - copy the snapshot from staged area to local repository
git reset - unstage a file
git restore - unstage a file or reset file in working directory
git checkout - reset file in working directory
Inspect & History
git status - show staged and unstaged files
git diff - show line by line diff
git log - lists commit history
git show - shows the commit and diff
git ls-files - list files in repository
File Operations
git clean - remove untracked files
git rm - remove tracked files
git mv - move/rename tracked files
Branching
git branch - creating/listing/deleting branches
git checkout - switch between branches
git switch - same as checkout
git stash - stack for moving uncommitted changes to temporary storage
git rebase - applies the commits of this branch on top of the target branch's head
git cherry-pick - copy commit from one branch to another
git tag - set a custom name to specific commit
Remote
git remote - remote repository
git push - pushing local changes to remote
git fetch - getting remote changes to local repo
